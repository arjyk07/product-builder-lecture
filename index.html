산과 강 AI 이미지 분류기

인공지능이 당신의 사진을 분석하여 산인지 강인지 구분해 드립니다.

서비스 소개

본 서비스는 최신 딥러닝 기술을 활용하여 산과 강의 이미지를 실시간으로 분류하는 웹 애플리케이션입니다. 사용자는 별도의 설치 없이 브라우저에서 바로 사진을 업로드하여 인공지능의 분석 결과를 확인할 수 있습니다.

복잡한 지형지물 사이에서 인공지능이 어떤 특징을 잡아내어 산과 강을 구분하는지 흥미로운 경험을 제공합니다.

어떻게 동작하나요?

이 웹사이트는 Google의 Teachable Machine 라이브러리와 TensorFlow.js를 기반으로 작동합니다. 사용자가 업로드한 이미지는 외부 서버로 전송되지 않고 브라우저 내부에서 즉시 처리되므로 개인정보가 안전하게 보호됩니다.

학습된 모델 정보

수천 장의 산과 강 이미지를 사전 학습(Pre-training)시킨 모델을 사용합니다. 이미지의 색상 분포, 질감, 수평선 패턴 등을 분석하여 확률론적인 결론을 도출합니다.
